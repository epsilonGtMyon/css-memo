行と列のレイアウト


.layout1 {
  display: grid;
  grid-template-columns: 100px 100px;
  grid-template-rows: 50px 50px;

  grid-auto-rows: minmax(20px, auto);
}

grid-template-columnsで列の幅
grid-template-rowsで行の高さを設定
columnsの方が優先度高そう
これだと5, 6は何も適用されていない

grid-auto-rows を使うと暗黙的な行, 5, 6に対して設定ができる。

配置１

        
demo2 .item1 {
  grid-column-start: 1;
  grid-column-end: 3;
}
demo2 .item2 {
  grid-row-start: 2;
  grid-row-end: 4;
}
        
item1, item2に対して線に対してアイテムの設置を設定
grid-column-start, grid-column-endはグリッドの左側を1とした線でどの位置までかを決める
同様にgrid-row-start, grid-row-endもグリッドの上側を1とした線で決める

割合でのサイズ指定

        
.layout2 {
  display: grid;
  grid-template-columns: 1fr 2fr;
  grid-template-rows: 1fr 1fr;
  
  width: 500px;
  height: 300px;
}
        
grid-template-columns, grid-template-rowsはfrという単位が利用できて、割合に応じたサイズ指定ができる。
上の指定だと1fr 2frとなってるので 1/3, 2/3がそれぞれ割り当てられる

サイズ指定のリピート

        

.layout3 {
  display: grid;
  grid-template-columns: 1fr repeat(2, 1fr 2fr) 1fr;
  grid-template-rows: repeat(3, 1fr);

  width: 1000px;
  height: 300px;
}
        
repeat()記法を使うことで 指定した回数分、サイズを指定することができる。

ギャップの指定

        
.layout4 {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: repeat(4, 1fr);

  column-gap: 10px;
  row-gap: 20px;
  
  width: 1000px;
  height: 300px;
}
        
column-gap, row-gapで行、列の間の間隔を指定できる。
